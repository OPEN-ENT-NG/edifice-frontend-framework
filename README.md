## Prerequisites

- `pnpm: >= 9 | 10`
- `node: >= 20 | 22`

## Getting Started

```bash
pnpm install
```

### Build

```bash
pnpm run build
```

### Eslint Lint

```bash
pnpm run lint
```

### Eslint Fix

```bash
pnpm run fix
```

### Prettier

```bash
pnpm run format
```
